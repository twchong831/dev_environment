# neovim in Mac

## install

- install neovim using brew

```bash
brew install neovim
```

### lua

```bash
brew install lua
```

### package tools

#### packer

```bash
mkdir -p ~/.local/share/nvim/site/pack/packer/start
cd ~/.local/share/nvim/site/pack/packer/start
git clone https://github.com/wbthomason/packer.nvim.git
```
